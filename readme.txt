1. About this project

	This is a fork of androminion by Ankur Metah, originated from google code
	project "androminion", an android based	implementation of the game
	"Dominion" by Rio Grande Games.
	
	This fork is currently based on version r628 (2012-08-29) of the official
	project.
	
	Goals of this fork are:
   - Implement new cards introduced in the Dark Ages expansion
   - Implement new game mechanics, including different setup at the start of the game, different new card types and their functionality, etc.
